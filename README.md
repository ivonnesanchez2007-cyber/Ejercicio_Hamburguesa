# 🍔 Sistema de Hamburguesa en Java

## 📖 Descripción
Aplicación en Java que permite crear una hamburguesa ingresando sus ingredientes por consola y mostrar su información.

## 🧠 Conceptos aplicados
- Programación Orientada a Objetos (POO)
- Encapsulamiento
- Uso de clases y objetos
- Getters y Setters
- Entrada de datos con Scanner

## 📂 Estructura
- Hamburguesa.java: Define los atributos y métodos
- App.java: Contiene el método main y la interacción con el usuario

## ⚙️ Ejecución
App.java
